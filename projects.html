Hyper Projects

Before Hyper I had no experience of coding. During our first 7 week course Data Landscape I learned so much, and when it was over I realised how much I loved it. The languages I was introduced to were HTML, CSS and JavaScript.

Looking back, I feel that I've made a lot of progress during my first time here at Hyper, and I've made sure to keep coding in my free time so that I can keep learning even more. Right now, I am reading this book. I am also planning to learn how to use node.js and how to animate with CSS soon.

Here are some of the sites I've made so far:





This was the second page I ever made, during my second week of coding. The assignment was to find some sort of data that we thought was interesting and then visualize it through graphs. We were also supposed to make it available online. To make these graphs, I used chart.js. It was quite challenging, seeing as I knew no JavaScript at all at this point, but I made it work anyway!

I made this site together with a classmate. This week, the challenge was to use an API. We found a Star Wars API (my partner was a massive fan) and somehow got it to work. I think it was a very useful experience, since I understood nothing from the beginning and thought it was going to be impossible to make it work, but we still did it! It's case sensitive though, but you can't have it all...

The final assignment of Data Landscape was to make a site offering some kind of service that incorporated everything we'd learned so far. My group and I created Adventuring.se: a site where you can generate a random adventure to go on somehwere in Stockholm. This is where I learned a lot of the JavaScript I know now. We used the Google directions API, created our own Json file with different locations and a script where you input your location to get a random location. We also made it so that you can choose a category to make sure that you get an adventure that suits your needs.
